    <article class="markdown-body entry-content" itemprop="text"><h1><a id="user-content-opentk-tutorials" class="anchor" aria-hidden="true" href="#opentk-tutorials"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>OpenTK Tutorials</h1>
<h3><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Introduction</h3>
<p>Basic and intermediate examples for getting started with <a href="https://opentk.github.io/" rel="nofollow">OpenTK</a> from C# and .Net.</p>
<p>C# projects "translated" from C++ available at the excellent <a href="http://www.opengl-tutorial.org" rel="nofollow"><strong>OpenGL tutorials site</strong></a>.</p>
<p>This is work in progress, the plan is to put all the examples from <em>opengl-tutorial</em> here, it might take a while though.</p>
<blockquote>
<p><strong>Although OpenTK supports multiple platforms, my code was only built and tested on Windows and Linux.</strong></p>
</blockquote>
<h3><a id="user-content-what-you-need" class="anchor" aria-hidden="true" href="#what-you-need"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>What you need</h3>
<ul>
<li>Of course, OpenTK. You can get it as a <a href="https://www.nuget.org/packages/OpenTK/" rel="nofollow">NuGet package</a> or build from <a href="https://github.com/opentk/opentk">source</a>. I use version 3.0.1 here.</li>
<li>A building environment (I use Visual Studio Express 2017 on Windows and build with mono on Linux).</li>
<li>A graphics card with drivers supporting OpenGL 3.3 or better. Almost everything newer than 3 or 4 years old should do. Sometimes a graphics driver update might be necessary. (If you aren´t sure about your card, check its capabilities with <a href="http://realtech-vr.com/admin/glview" rel="nofollow">OpenGL Extensions Viewer</a> on Windows or using <code>gfxinfo</code> on Linux.)</li>
<li>Some experience with C#. The examples are not meant to introduce you to a new language...</li>
</ul>
<h3><a id="user-content-differences-from-opengl-tutorials" class="anchor" aria-hidden="true" href="#differences-from-opengl-tutorials"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Differences from OpenGL tutorials</h3>
<p>I mentioned above that these tutorials are heavily based on the ones on <a href="http://www.opengl-tutorial.org" rel="nofollow"><strong>OpenGL tutorials site</strong></a>.</p>
<p>If you peruse that site you´ll find excellent introductory texts to OpenGL features, apart from example code.</p>
<p>I kept the same basic organization they use (i. e. the tutorials have the same numbers). The recommended use is to start with their text and then read mine, which will only contain additional details relevant to OpenTK.</p>
<h3><a id="user-content-structure" class="anchor" aria-hidden="true" href="#structure"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Structure</h3>
<p>Every tutorial is a standalone project in order to make it easier to understand and build each example. All tutorials are part of an overarching solution.</p>
<p>Each projects contains all files necessary to build (textures, shaders etc.).</p>
<h3><a id="user-content-index" class="anchor" aria-hidden="true" href="#index"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Index</h3>
